How It Works
  1. Upload PDF: Select the supplier and upload the BOL PDF file
  2. Auto-Extraction: pdfPlumber extracts text and tables from the PDF
  3. Regex Parsing: The system uses regex patterns to identify and extract key fields:
    • Bill Number
    • Shipper and Consignee information
    • Origin and Destination
    • Carrier, Vessel, Container details
    • Line items (products, quantities, weights)
    • Dates and charges
  4. Review: You'll see a form pre-filled with the extracted data
  5. Correct: Review and correct any mistakes or missing information
  6. Save: Create the BOL in the database as a structured record
Tips for Best Results
  • Use clear, well-formatted PDF files
  • Ensure text is selectable (not scanned images)
  • Standard BOL formats work best
  • File size limit: 15MB
  • Only PDF files are accepted
Alternative Methods

If PDF extraction doesn't work well for your BOL format, you can:

  • Create BOL manually
  • Contact support to improve extraction for your format
